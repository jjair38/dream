# Dream Burguer - Landing Page Premium

Este é o projeto da Landing Page da **Dream Burguer**, uma lancheira e hot-dogaria artesanal localizada em Jarinu/SP. 

O projeto foi desenvolvido para oferecer uma experiência de usuário premium, com foco em conversão para pedidos via WhatsApp e visualização de cardápio.

## 🚀 Tecnologias e Estilo

-   **Next.js 15+**: Framework moderno para alta performance.
-   **Tailwind CSS**: Estilização via utilitários.
-   **Framer Motion**: Animações fluidas e premium.
-   **Lucide Icons**: Conjunto de ícones consistentes.

## 📁 Estrutura de Arquivos

-   `app/page.tsx`: Componente principal da aplicação.
-   `app/layout.tsx`: Configurações de metadados (título e descrição do site).
-   `app/globals.css`: Estilizações globais e variáveis de tema.
-   `next.config.ts`: Configurações do Next.js (permissões de imagens externas).

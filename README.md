# Dream Burguer - Landing Page Premium

Este é o projeto da Landing Page da **Dream Burguer**, uma lancheira e hot-dogaria artesanal localizada em Jarinu/SP. 

O projeto foi desenvolvido para oferecer uma experiência de usuário premium, com foco em conversão para pedidos via WhatsApp e visualização de cardápio.

## 🚀 Tecnologias e Versões

O projeto contém duas versões principais:

1.  **Versão React/Next.js (Moderna):**
    -   Utiliza **Next.js 15**, **Tailwind CSS** e **Framer Motion** para animações fluidas.
    -   Ideal para hospedagem em plataformas como Vercel ou Netlify.
    -   Código localizado em `/app` e configurado no `package.json`.

2.  **Versão PHP/HTML (Estática/Hospedagem Simples):**
    -   Arquivo único `index.php`.
    -   Utiliza **Tailwind CSS via CDN** e **Lucide Icons**.
    -   Ideal para servidores de hospedagem tradicionais (Ex: Hostgator, Locaweb, etc).

## 📁 Estrutura de Arquivos

-   `index.php`: Versão completa em PHP/HTML para implantação rápida.
-   `app/page.tsx`: Componente principal da versão Next.js.
-   `app/globals.css`: Estilizações globais e variáveis de tema.
-   `next.config.ts`: Configurações do Next.js (incluindo permissões de imagens externas).
